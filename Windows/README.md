# WINDOWS LABS

Hands-on labs covering Windows 11 VM environment setup, creating & restoring snapshots, account 
administration, installing and uninstalling software, file permissions, windows updates and system diagnostics.

| Lab | Description | Skills Demonstrated |
|---|---|---|
| [VM Setup & Snapshots](01-win11-vm-setup-snapshots/) | Built a Windows 11 VM and established/tested a baseline snapshot | VM provisioning, snapshot management |
| [Local Accounts & Permissions](02-local-user-account-administration/) | Created and managed local accounts, configured file permissions | User administration, NTFS permissions |
| [Software Management](03-software-management/) | Installed and cleanly uninstalled applications | Software deployment/removal |
| [Windows Updates](04-windows-updates/) | Managed and troubleshot Windows Update | Patch management |
| [System Diagnostics](05-system-diagnostics-toolkit/) | Diagnosed an issue using Event Viewer, Task Manager, Services, and related tools | System diagnostics, root-cause analysis |
