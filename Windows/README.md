# Windows Labs

Hands-on labs covering Windows 11 VM environment setup, creating and restoring snapshots, account 
administration, installing and uninstalling software, file permissions, Windows Update and system diagnostics.
| Lab | Description | 
|---|---|
| [Win 11 VM Setup & Snapshots](01-win11-vm-setup-snapshots/) | Built a Windows 11 VM and established/tested a baseline snapshot | 
| [Local Accounts & File Permissions](02-local-user-account-administration/) | Created and managed local accounts, configured file permissions |
| [Software Management](03-software-management/) | Installation and uninstallation of applications | 
| [Windows Updates](04-windows-updates/) | Managed and troubleshot Windows Update |
| [System Diagnostics](05-system-diagnostics-toolkit/) | Diagnosed an issue using Event Viewer, Task Manager, Services, and related tools |
