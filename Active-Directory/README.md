# Windows Server Labs

Hands-on labs covering the deployment and administration of Windows Server 2025, from standalone baseline configuration to services integrated with an Active Directory domain. The environment is shared with the related Active Directory labs, allowing server roles and access controls to be tested from a domain-joined Windows 11 client.

| Lab | Description | 
|---|---|
| [Windows Server Deployment & Baseline Configuration](01-server-deployment-baseline/) | Deployed a Windows Server 2025 VM, configured the hostname, static network settings, updates and connectivity, and created a baseline snapshot |
| [DHCP Server Deployment & Client Configuration](02-dhcp-server-deployment/) | Installed and authorized DHCP, configured a scope and network options, and verified address assignment on a domain-joined Windows 11 client |
| [File Services, Permissions & Recovery](03-file-services-permissions-recovery/) | Created SMB shares, assigned share and NTFS permissions through Active Directory security groups, tested user access, and backed up and restored server data |
